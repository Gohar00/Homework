and - տրամաբանական "և" օպերատոր
break - ցիկլի ընդհատում
def - ֆունկցիայի սահմանում
del - օբյեկտի հեռացում
elif - օգտագործվում է պայմանական հրամանում
for - ստեղծում է ցիկլ
if - ստեղծում է պայմանի հրաման
is - ստուգում է,արդյոք երկու փոփոխականներ հավասար են
in - ստուգում է,արդյոք փոփոխականը կա լիստի մեջ կամ մեկ այլ զանգվածում
None - ներկայացվում է զրայական արժեք
or - տրամաբանական օպերատոր
return - ավարտում է ֆունկցիան և վերադարձնում արժեք
True - բուլյան արժեք, համեմատական գործողությունների արդյունք,երբ ճիշտ է
while - ստեղծում է ցիկլ
False - բուլյան արժեք, համեմատական գործողությունների արդյունք,երբ սխալ է
